# WriteAway

**Keep typing or watch your words disappear.**

WriteAway is a focused writing app that helps you maintain momentum and overcome writer's block. The concept is simple: keep typing or your text starts to fade away, character by character.

---

## âœ¨ Features

- **Three Writing Modes:**
  - Normal: Visual warnings before deletion
  - Gentle: Text fades to gray first
  - Hardcore: Immediate deletion with minimal grace period

- **Customizable Sessions:**
  - Preset durations: 5, 10, 15, 30 minutes
  - Adjustable grace period (3-15 seconds)
  - Variable deletion speeds

- **Writing-Focused Interface:**
  - Dark/light themes
  - RTL/LTR text direction support
  - Fullscreen mode
  - Adjustable text size and fonts
  - Distraction-free writing environment

- **Session Statistics:**
  - Word count tracking
  - Peak word count
  - Longest typing streak
  - Characters survived
